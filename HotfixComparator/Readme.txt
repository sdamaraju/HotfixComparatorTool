# HotfixComparator

Hotfix-Comparator - as the name suggests, this tool does a full compare on the hot-fix files taken from 2 different environments from the same PRPC version.
No more manual efforts for comparing the hotfix scan sheets, just upload them to this tool and you have the results right away. 

## Deployment and usage instructions

1.Download the jar file and .bat file and place them under the same folder structure.
2.Run the .bat file.
3.On the Browser, just hit the : http://localhost:8080
4.Upload the csv files on to the browser.
5.Click the Compare Button.

### Prerequisites

Oracle JDK 1.8
