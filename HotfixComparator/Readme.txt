# HotfixComparator

Hotfix-Comparator - as the name suggests, does a full compare on the hot-fix files taken from 2 different environments from the same PRPC version.
No more manual efforts for comparing the hotfix scan sheets, just upload them to this tool and you have the results right away. 

## Getting Started

1.Download the jar file and .bat file and run the .bat file.
2.On the Browser, just hit the  : http://localhost:8080
3.Upload the csv files on to the browser.

### Prerequisites

Oracle JDK 1.8
